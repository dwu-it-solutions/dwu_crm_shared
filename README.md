# DWU CRM Shared

Tipos e utils compartilhados - @dwu/crm-shared

## Estrutura do Projeto

```
src/
├── types/
├── enums/
├── validations/
├── constants/
└── utils/
```

## Instalação

```bash
npm install @dwu/crm-shared
```

## Desenvolvimento

```bash
npm install
npm run build
```

## Branches

- `main` - Produção
- `staging` - Pré-produção
- `development` - Desenvolvimento
